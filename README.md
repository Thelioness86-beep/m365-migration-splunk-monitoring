This project demonstrates how Splunk can be used to monitor a Microsoft 365 migration environment, including mailbox migration status, endpoint compliance, IAM access issues, Conditional Access failures, endpoint security alerts, and post-migration support trends.
The project uses sample JSON log data and mock dashboards to demonstrate monitoring, analysis, alerting, and governance reporting for a cloud migration team.
